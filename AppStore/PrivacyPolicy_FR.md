# Politique de Confidentialité - CourrierCode

**Dernière mise à jour : 15 janvier 2026**

## Introduction

Bienvenue sur CourrierCode. Cette politique de confidentialité explique comment nous traitons vos informations lorsque vous utilisez notre application.

## Résumé

**CourrierCode ne collecte, ne stocke et ne partage AUCUNE donnée personnelle.**

## Données collectées

### Données que nous NE collectons PAS :

- ❌ Informations personnelles (nom, email, téléphone)
- ❌ Données de localisation
- ❌ Contacts ou carnet d'adresses
- ❌ Photos ou fichiers
- ❌ Messages encodés ou décodés
- ❌ Codes secrets utilisés
- ❌ Historique d'utilisation
- ❌ Identifiants publicitaires
- ❌ Données de navigation
- ❌ Informations sur l'appareil

### Fonctionnement local

Toutes les opérations de CourrierCode sont effectuées **localement sur votre appareil** :

- L'encodage et le décodage des messages se font entièrement sur votre iPhone/iPad
- Les codes secrets et tables aléatoires ne quittent jamais votre appareil
- Aucune connexion Internet n'est requise pour utiliser l'application
- Aucune donnée n'est envoyée à nos serveurs ou à des tiers

### Stockage local

Les seules données stockées localement sur votre appareil (via UserDefaults) sont :

- Votre code secret pour déverrouiller la table de référence
- Le code de la table aléatoire si vous en avez activé une

Ces données restent exclusivement sur votre appareil et ne sont jamais transmises.

## Services tiers

CourrierCode n'intègre aucun service tiers :

- ❌ Pas de publicité
- ❌ Pas d'analytics
- ❌ Pas de SDK de suivi
- ❌ Pas de réseaux sociaux intégrés
- ❌ Pas de services cloud

## Sécurité

Bien que nous ne collections aucune donnée, nous prenons la sécurité au sérieux :

- L'application fonctionne entièrement hors ligne
- Aucune transmission de données sur Internet
- Code source développé selon les meilleures pratiques iOS

## Enfants

CourrierCode est adapté à tous les âges (4+) et ne collecte aucune information sur les enfants ou les adultes.
